# 功能说明

Task DAG 用一张依赖图管理一个 Goal。节点表示目标、项目或任务；连线表示“子节点/前序步骤实现父节点”。

## 图语义

- 根节点固定为 Goal。
- 直连 Goal 的节点显示为 Project。
- 更深层节点显示为 Task。
- 边方向固定为 `from = 子节点 / 前序步骤`，`to = 父节点 / 被实现的节点`。
- 创建或拖线形成环时会被拒绝。

## 节点状态

- `todo`：待开始。
- `doing`：进行中。
- `done`：已完成。
- `paused` 链条：节点和相关边变灰，前序子树折叠。
- 可开始：节点处于 `todo`，且所有前序步骤均为 `done` 时显示金色高亮。
- 截止日期：未完成且过期的节点显示红色日期角标。

## Goal 管理

工具栏左侧菜单支持：

- 切换 Goal。
- 新建 Goal。
- 删除当前 Goal，至少保留一个。
- 重命名当前 Goal。
- 导出当前 Goal JSON。
- 导入 JSON 为新 Goal，不覆盖现有 Goal。
- 绑定或解除本地数据目录。

## 节点操作

### 键盘

| 键 | 行为 |
|---|---|
| `Tab` | 在选中节点后创建后继任务，并自动连线 |
| `Enter` | 创建同级并行任务，继承选中节点的父级连接 |
| 双击 / `F2` | 编辑节点标题或打开详情面板 |
| `Space` | 切换状态：待开始 → 进行中 → 已完成 |
| `D` | 打开节点详情面板 |
| `Delete` / `Backspace` | 删除选中节点或选中边 |
| 方向键 | 沿依赖线或同列节点移动选中 |
| `Ctrl/Command+Z` | 撤销 |
| `Ctrl/Command+Shift+Z` / `Ctrl+Y` | 重做 |
| `Esc` | 取消选中、取消编辑或关闭浮层 |

输入框、文本框、下拉框聚焦时，画布快捷键不会抢占输入。

### 鼠标

- 单击节点选中。
- Command/Ctrl + 单击可多选节点。
- 双击节点打开详情面板。
- 拖动树内节点调整同父节点上下顺序。
- 把树内节点拖到另一个节点附近可替换父连接。
- 从连接点拖到另一个节点可创建依赖边。
- 右键边可删除依赖。
- 拖动画布空白处平移。
- 滚轮缩放。
- 双击空白处创建游离任务。

## 详情与样式

节点详情面板支持：

- 标题。
- 描述。
- 链条状态。
- 任务状态。
- 预计工时。
- 截止日期。

边详情面板支持：

- 从固定色板设置边颜色。
- 重置边颜色。

## 布局

- 默认使用自动布局，子节点位于父节点右侧。
- 可切换左右展示方向。
- 游离任务保留手动位置。
- 连入图中的节点回到自动布局。
- “整理布局”会清除手动位置，并尽量保留当前同层上下顺序。

## 数据存储

- 默认保存到浏览器 `localStorage`。
- `localhost` 和 `file://` 是不同浏览器来源，因此 localStorage 数据不同。
- Chrome/Edge 可绑定一个本地目录，把每个 Goal 保存为独立 JSON 文件。
- 绑定目录后，切回标签页时会扫描目录变更并刷新当前 store。
- 目录同步面向单用户本地使用，不提供多人冲突解决。

## 导入导出

导出内容是当前 Goal 的 JSON。导入时会先按当前 schema 迁移和校验，合法后作为新 Goal 加入；非法 JSON 不会覆盖现有数据。
